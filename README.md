## AUTHFORGE :
AuthForge is a modern authentication and user management system built from scratch using Node.js, Express.js, Javascript, HTML and CSS.

## CURRENT MODULE:
 -- User Management Module (CRUD Complete)
## Features:
 -User Registration
 -View Registered Users(GET)
 -Update User (PUT)
 -DELETE User (DELETE)
 -Admin Dashboard
 -JSON-based data storage
 -Password Hashing with bcrypt
 -Secure Password Storage
 -Restful API Architecture
 

## Tech Stack:
