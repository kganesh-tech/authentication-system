## AUTHFORGE :
AuthForge is a modern authentication and user management system built from scratch using Node.js, Express.js, Javascript, HTML and CSS.

## CURRENT MODULE:
 -- User Management Module
## Features:
 -User Registration
 -View Registered Users(GET)
 -Admin Dashboard
 -JSON-based data storage

## Tech Stack:
-HTML
-CSS
-JavaScript
-Node.js
-Express.js

## Roadmap:
-Update User(PUT)
-Delete User(DELETE)
-Password Hashing(bcrypt)
-User Login
-JWT Authentication
-MongoDB
-Role-Based Access Control
-API Key Management
